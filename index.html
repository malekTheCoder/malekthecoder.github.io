Voice-Controlled Robotics Agent

A fully voice-controlled robotic manipulation agent running in MuJoCo physics simulation. Speak a natural-language command (like "stack the red cube on the blue cylinder") and a Franka Panda arm plans and executes it in real time. An LLM performs task decomposition into ordered sub-goals via chain-of-thought reasoning, detects mid-task failures, and triggers agentic replanning to recover. Voice commands are transcribed via a speech-to-text API and responses spoken back via a text-to-speech API. A second independent LLM call verifies the live scene against the goal state before reporting success. Robot motion is solved via inverse kinematics using the mink library. Live simulation streams through a FastAPI WebSocket portal.
